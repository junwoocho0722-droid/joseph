# 학교에서 사용해보기 안내

이 앱은 학교 보건실 물품 재고 관리 전용 앱입니다.
학생 이름, 증상, 질병, 상담 내용, 개인 건강 정보는 입력하지 마세요.

## 가장 쉬운 방법

1. `school_health_room_inventory_app.zip` 파일을 USB나 이메일로 학교 PC에 가져갑니다.
2. 학교 PC에서 압축을 풉니다.
3. 학교 PC에 Python이 없다면 먼저 설치합니다.
   - https://www.python.org/downloads/
   - 설치할 때 `Add Python to PATH` 옵션을 체크하는 것이 좋습니다.
4. 압축을 푼 폴더에서 `start_app.bat` 파일을 더블클릭합니다.
5. 브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:8501
```

## 휴대폰이나 태블릿에서 접속하려면

1. 학교 PC와 휴대폰이 같은 Wi-Fi에 연결되어 있어야 합니다.
2. 앱을 실행한 PC의 IP 주소를 확인합니다.
3. 휴대폰 브라우저에서 아래처럼 접속합니다.

```text
http://PC의_IP주소:8501
```

예:

```text
http://192.168.0.10:8501
```

학교 Wi-Fi에서 기기 간 접속을 막아두었거나 Windows 방화벽이 막고 있으면 휴대폰 접속이 안 될 수 있습니다.

## 데이터 저장 위치

입력한 재고 데이터는 같은 폴더의 `health_room_inventory.db` 파일에 저장됩니다.
학교 PC에서 새로 시작하면 새 데이터베이스가 만들어집니다.

현재 이 PC에서 입력한 데이터를 그대로 가져가려면 `health_room_inventory.db` 파일도 함께 복사하세요.

## 앱을 종료하려면

앱 실행 창에서 `Ctrl + C`를 누르거나 창을 닫으면 됩니다.
