# 학교 보건실 재고 관리 앱

학교 보건실의 의약품과 응급처치 용품 재고를 관리하는 간단한 Streamlit 앱입니다.

이 앱은 재고 관리 전용입니다. 의학적 조언을 제공하거나 의약품을 추천하지 않으며, 학생 이름, 증상, 질병, 개인 건강 정보를 저장하지 않습니다.

## 주요 기능

- 재고 물품 추가
- 현황판에서 전체 물품 수 확인
- 부족 물품, 유효기간 임박 물품, 유효기간 지난 물품 알림 확인
- 날짜와 수량만 사용 기록으로 저장
- 기존 물품에 재고 추가
- 재고와 사용 기록을 CSV 파일로 내보내기

## 실행 방법

필요한 패키지를 설치합니다.

```bash
pip install -r requirements.txt
```

앱을 실행합니다.

```bash
streamlit run app.py
```

앱 데이터는 `health_room_inventory.db`라는 로컬 SQLite 파일에 저장됩니다.
